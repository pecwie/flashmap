# Installation

## Stable release

To install flashmap, run this command in your terminal:

```
pip install flashmap
```

This is the preferred method to install flashmap, as it will always install the most recent stable release.

If you don't have [pip](https://pip.pypa.io) installed, this [Python installation guide](http://docs.python-guide.org/en/latest/starting/installation/) can guide you through the process.

## From sources

To install flashmap from sources, run this command in your terminal:

```
pip install git+https://github.com/pecwie/flashmap
```
